# sai-boe-dogc-bot
Automatització SAI per al monitoratge del DOGC i BOE. Detecta convocatòries rellevants per a empreses (R+D, indústria, energia, digitalització, internacionalització), descarrega automàticament les bases i la convocatòria en PDF, genera un resum estructurat i envia el correu intern amb els documents adjunts.
